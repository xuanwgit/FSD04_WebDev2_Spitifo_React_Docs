Spitifo Music App

A modern music streaming platform built with the MERN stack

About the Application

Spitifo is a sophisticated music streaming platform that combines modern web technologies with an intuitive user interface. Built using the MERN stack (MongoDB, Express.js, React, Node.js), it provides users with a seamless music streaming experience while offering robust features for managing and sharing music collections.

Core Features

User Authentication

  • Secure signup and login
  • JWT-based authentication
  • Password encryption
  • Protected routes

Album Management

  • Create and edit albums
  • Upload album artwork
  • Manage visibility settings
  • Add and remove tracks

Music Player

  • Stream audio tracks
  • Playlist management
  • Shuffle and repeat options
  • Progress tracking

Technical Stack

Frontend

React.js

Backend

Node.js & Express

Database

MongoDB

Storage

AWS S3
